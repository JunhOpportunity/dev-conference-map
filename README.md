# dev-conference-map
MAP API를 활용해 국내 개발자 컨퍼런스를 확인할 수 있는 웹 프로젝트

## 개발 환경
### 공통
- VSCODE
### 프론트엔드
- HTML/CSS/JAVASCRIPT
- REACT 18
- Redux toolkit
### 백엔드
- maria DB
- java 17
- spring boot 3.3.6
